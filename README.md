# Структура файлов проекта:
* `stream4_handson_project.ipynb` - основной блокнот проекта данного спринта №4
* `scripts/` - папка с модулями питон (в которые вынесен весь громоздкий код для повышения читаемости блокнота проекта):
  * `dataset.py`с функциями для подготовки пайплайна даных
  * `utils.py` с функциями для обучения моделей и визуализации результатов
* `cfg/` - папка с сейвами удачных обученных моделей
* `data/` - папка с исходным датасетом (не копируется в репозитарий)
* `models/` - папка с загруженой моделью семантического эмбеддинга текстов (Word2Vec)
* `lessons_notebook/` - папка с примерами и задачами уроков теорчасти спринта №4 (не используется напрямую в проекте, но при ее заполнении собиралось виртуальное окружение, в котором работает код проекта)
* `pyproject.toml` - файл зависимостей виртуального окружения, в котором работает код проекта

# Описание проекта:

## Цели проекта:
* собрать мультимодальную DL-модель, которая по NLP и CV фичам объектов (состав и картинки блюд) будет предсказывать числовую характеристику объекта
* достичь точности обучения с МАЕ < 50 на тестовых данных
* визуализировать и проанализировать результаты

## Что сделано в проекте:
* построены baseline модели на ML-векторизации панельных данных о блюдах:
  * линейная регрессия
  * HGBoost ансамбли решающих деревьев
  * MLP NN регрессор
* в качестве подхода к мультимодальности выбрано использовать предобученные модели из доступных библиотек для генерации DL-фич объектов двух групп - текстовые эмбеддинги и cv-features сверточной части сети MobileNet
* полученные фичи сконкатенировать в один вектор, добавить массу блюд и обучить поверх этих данных MLP NN с дропаутом и нормализацией батчей
* поскольку дообучение сверточной части модели не планировалось, аугментация изображений в пайплайне не понадобилась - предполагалось, что выходные cv-фичи уже устойчивы к аугментации изображений
* при обучении моделей достигнута следующая точность:
  * `linear regres ML` MAE  = 56.5
  * `HXBoost model ML` MAE  = 51.7
  * `best baseline ML` MAE  = 37.8
  * `best  multimodal` MAE  = 46.1
  * `avg best  models` MAE  = 35.5  
* обнаружена сильная семантическая взаимосвязь ML-векторизации с целевой переменной, что позволило baseline модели обогнать по точности мультимодальную модель, однако обе модели оценивают объекты с принципиально разных сторон, поэтому простым усреднением их прогнозов получена еще более точная модель, которая превзошла каждую из составляющих моделей в отдельности
* визуально проанализированы самые точные и самые неточные прогнозы - сделаны выводы-гипотезы о возможных причинах
